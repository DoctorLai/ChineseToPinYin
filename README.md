# API Example
```
https://helloacm.com/api/pinyin/?cached&s=小赖子英国生活和资讯JustYY.com&t=1
```
This returns JSON data:
```
{"result":["xiao3","lai4","zi3","ying1","guo2","sheng1","huo2","he2,he4","zi1","xun4","J","u","s","t","Y","Y",".","c","o","m"]}
```
